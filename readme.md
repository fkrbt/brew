# brew
basically just a web app made for wither where you can watch movies, tv shows, and live tv.

how to run:

download visual studio code

make a new folder and open it with vscode

in bash terminal, do these things:

`git clone https://https://github.com/fkrbt/brew
cd brew
python3 -m http.server 8080`

and youre done! go to localhost:8080 to use the site!
